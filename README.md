# EPOS Management System

Enterprise Point of Sales and Inventory Management System built using Spring Boot and Laravel.

---

## Features

- JWT Authentication
- Role Based Access Control
- Product Management
- Inventory Management
- Sales Transaction
- Dashboard Analytics
- Reporting System
- Real-time Notification (planned)

---

## Tech Stack

### Backend
- Java 17
- Spring Boot 3
- Spring Security
- JWT Authentication
- PostgreSQL
- JPA / Hibernate

### Frontend
- Laravel
- Blade Template
- Bootstrap / TailwindCSS

### Infrastructure
- Docker
- Redis (planned)
